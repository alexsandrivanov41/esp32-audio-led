## 📖 Описание проекта

Прошивка для **ESP32-WROVER-B Audio Streamer v3.0** — многофункциональное устройство для потоковой передачи аудио с синхронизированной светодиодной подсветкой.

### 🔑 Ключевые возможности

- **Аудио стриминг**: Приём и воспроизведение аудио через I2S (DAC)
- **RGB LED контроль**: Управление адресными светодиодами (WS2812/SK6812)
- **Синхронизация**: Аудио-визуальные эффекты в реальном времени
- **Веб-интерфейс**: Встроенный HTTP-сервер и WebSocket для управления
- **Безопасность**: AES-шифрование трафика, SHA-256 аутентификация
- **Энергонезависимая память**: Сохранение настроек в EEPROM
- **Режимы работы**: 
  - Стриминг аудио с внешнего источника
  - Генерация тестовых сигналов
  - Световые шоу (статичные цвета, радуга, реакция на звук)

## 🛠 Технические характеристики

### Оборудование
- **Платформа**: ESP32-WROVER-B (Dual Core, 240 MHz, 4MB Flash, 8MB PSRAM)
- **Аудио интерфейс**: I2S (Input/Output)
- **LED интерфейс**: RMT (Remote Control) + LED_strip
- **Wi-Fi**: STA/AP режимы, поддержка WPA2


## 🚀 Быстрый старт

### Требования
- Arduino IDE 2.x или PlatformIO
- Плата ESP32-WROVER-B
- Подключенные периферийные устройства (DAC, LED лента)

### Установка зависимостей
1. Установите ядро ESP32 в Arduino IDE:
   - Меню → Файл → Настройки → Дополнительные URL для плат:
     ```
     https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
     ```
   - Инструменты → Платы → Диспетчер плат → Поиск "ESP32" → Установить

2. Установите библиотеки:
   - `WebSockets` by Markus Sattler
   - `FastLED` или `Adafruit NeoPixel` (опционально)

### Прошивка
1. Откройте `main.ino` в Arduino IDE
2. Выберите плату: **Tools → Board → ESP32 Dev Module**
3. Настройте параметры:
   - Upload Speed: `921600`
   - CPU Frequency: `240MHz`
   - Flash Size: `4MB`
   - PSRAM: `Enabled`
4. Подключите ESP32 по USB
5. Нажмите **Upload**

### Первоначальная настройка
После прошивки устройство создаёт точку доступа:
- **SSID**: `AudioStreamer_XXXX`
- **IP**: `192.168.4.1`
- **Веб-интерфейс**: `http://192.168.4.1`

Через веб-интерфейс можно:
- Подключиться к домашней WiFi сети
- Настроить яркость и режимы LED
- Запустить тест аудио/света

## 🔐 Безопасность

### Шифрование трафика
- **Алгоритм**: AES-128-CBC
- **Хеширование**: SHA-256 для аутентификации
- **Генерация ключей**: На основе PSK + случайного salt

### Процесс аутентификации
1. Клиент отправляет запрос на подключение
2. Сервер генерирует случайный challenge
3. Клиент вычисляет HMAC-SHA256(token + challenge)
4. Сервер проверяет подпись и предоставляет сессионный ключ

## 🎨 Режимы LED эффектов

| Режим | Описание |
|-------|----------|
| `STATIC` | Статичный цвет (настраиваемый) |
| `RAINBOW` | Плавная радужная анимация |
| `BREATH` | Эффект "дыхания" (затухание/разгорание) |
| `AUDIO_REACTIVE` | Реакция на амплитуду аудио сигнала |
| `VU_METER` | Визуализатор уровня громкости |
| `STROBE` | Стробоскоп по ритму |
| `OFF` | Все светодиоды выключены |

### Распространённые проблемы

| Проблема | Решение |
|----------|---------|
| Нет звука | Проверьте подключение I2S пинов, наличие питания DAC |
| LED не горят | Убедитесь в правильности направления DATA линии, проверьте питание 5V |
| Обрывы связи | Увеличьте мощность передатчика, проверьте антенну |
| Перезагрузки | Проверьте потребление тока, возможно недостаточно питания |
